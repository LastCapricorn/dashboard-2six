# dashboard-2six

The 2nd Project from [The Odin Project](https://www.theodinproject.com)'s Full Stack JavaScript Path.

That was an amazing adventure and finally this "Admin Dashboard" does everything I had in mind before - a lot more than expected for this project, but that's a quirk of mine...

## Things you can find on this *Dashboard*

- I tried to make it responsive.
  - check out different window sizes and emulate devices in developer tools

- Yes, the top-left logo is also a button

- You can start a google search

- The page should start with the user's preferred color scheme
  - You can change this
  - even though my choice of colors isn't that perfect

- Hidden elements open the corresponding menu when they receive focus and shouldn't affect the layout.

- You can check out some of my other TOP projects (live previews & repos)
  - among others the old *dashboard* from 2023
  - I think this new version seems more mature and less like candy - despite some of my shenanigans

- I played around with *corner-shape*, not yet supported by all browsers (e.g. Firefox, Safari)
  - Hopefully, they will do that soon
